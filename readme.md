Install Astrodon's cli:

```shell
deno install -A --unstable https://github.com/astrodon/astrodon/blob/feature/deno_tauri/modules/astrodon-cli/mod.ts
```

Run the app in development mode:

```
astrodon run
```

Build an executable:

```
astrodon build
```
